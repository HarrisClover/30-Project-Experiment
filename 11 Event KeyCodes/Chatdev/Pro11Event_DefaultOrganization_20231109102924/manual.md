# Keyboard Event Viewer

The Keyboard Event Viewer is a website that allows you to read the keyCodes of the keyboard and display the event details on the page. This user manual will guide you through the installation process and explain how to use the software.

## Installation

To use the Keyboard Event Viewer, you need to follow these steps:

1. Download the code files from the provided source.
2. Open the downloaded folder and locate the `index.html` and `script.js` files.

## Usage

Once you have installed the Keyboard Event Viewer, you can start using it by following these steps:

1. Open the `index.html` file in a web browser.
2. You will see a heading "Keyboard Event Viewer" and a paragraph "Press any key to see the event details".
3. Press any key on your keyboard.
4. The event details, including `event.key`, `event.keyCode`, and `event.code`, will be displayed below the paragraph.

## Example

Here is an example of how the Keyboard Event Viewer works:

1. Open the `index.html` file in a web browser.
2. Press the "A" key on your keyboard.
3. The event details will be displayed as follows:

```
event.key: A
event.keyCode: 65
event.code: KeyA
```

## Conclusion

The Keyboard Event Viewer is a simple and easy-to-use website that allows you to view the event details of the keys pressed on your keyboard. By following the installation and usage instructions provided in this manual, you will be able to use the software effectively. Enjoy exploring the event details of your keyboard!
