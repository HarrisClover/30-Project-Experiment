FAQ Section

What is an operating system?

▼

An operating system (OS) is system software that manages computer hardware, software resources, and provides various services for computer programs.

What is the difference between RAM and ROM?

▼

RAM (Random Access Memory) is volatile memory, it loses the information when the power goes off. ROM (Read Only Memory) is non-volatile memory, it does not lose information when power is lost.

What is the function of a processor in a computer?

▼

The processor (CPU) is the "brain" of the computer. It carries out the instructions of a computer program by performing basic arithmetical, logical, control and input/output (I/O) operations.
